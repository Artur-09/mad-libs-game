# Mad Libs Game

## Description

This is a simple Mad Libs game written in Python.
The user selects a story template and enters words such as nouns, adjectives, and numbers.
The program then generates a funny story using the user's inputs.

## Features

- Three different story templates:
  - Hospital Story
  - Camping Story
  - Castle Story
- Input validation:
  - Prevents empty input
  - Ensures numbers are numeric
  - Validates menu selection

## How to Run

1. Make sure Python is installed.
2. Download or copy the project files.
3. Open terminal or command prompt.
4. Run:

python mad_libs.py

5. Follow the instructions on the screen.

## Requirements

- Python 3.x
